## 注意事项
1. 此文章未涉及具体安装过程，如需了解，点击这里 查看；
2. 已修改 SSDT，所以如果要用此EFI，请按照这个配置来组装，特别是 主板、cpu、硬盘；
3. 已映射USB端口，网卡蓝牙连接线请插入F_USB2口（已映射为内置），不然会引起睡眠秒醒;
4. 已更换三码，也可以自已重新填入;
5. opencore版本为 OpenCore-0.7.5-RELEASE;
